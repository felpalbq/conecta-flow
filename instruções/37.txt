# Claude Code Project Setup
Version: 1.0

---

# Objetivo

Definir a preparação inicial do ambiente de desenvolvimento do Conecta Flow utilizando Claude Code como principal agente de construção.

Este documento descreve:

- preparação do repositório;
- organização inicial;
- responsabilidades do Claude Code;
- conexão com ferramentas;
- sequência inicial de trabalho.

---

# Filosofia

Claude Code deve atuar como engenheiro principal do projeto.

Ele não deve simplesmente gerar código.

Ele deve:

- compreender o domínio;
- respeitar arquitetura;
- propor soluções;
- documentar decisões;
- implementar incrementalmente.
